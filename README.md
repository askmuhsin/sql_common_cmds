### Create & Delete db
Check available dbs
```
SHOW DATABASES;
```
Create dbs
```
CREATE DATABASE db_name;
```
Delete dbs
```
DROP DATABASE db_name;
```
Access db
```
USE db_name;
```
See overview of the tables
```
SHOW tables;
```
### db queries
```
SELECT * FROM table;
```
